pyenv配置

  1. 安装git yum -y install git
  2. 安装pyenv curl -L https://raw.githubusercontent.com/yyuu/pyenv-installer/master/bin/pyenv-installer | bash

  3. 配置环境变量，在~/.bash_profile里增加。

  4. 安装编译工具yum -y install gcc make patch
